Resources | Software

Apps

Atom Text Editor
A hackable, cross-platform, open-source text editor by GitHub.
FFmpeg

A complete, cross-platform solution to record, convert and stream audio and video.

(for Mac users: download the zip, unzip, and copy ffmpeg to /usr/local/share)

GIMP
The GNU Image Manipulation Program is a free and open-source cross-platform raster graphics editor.
Git for Windows
A lightweight, native set of tools that brings the full feature set of the Git SCM to Windows while providing appropriate user interfaces for experienced Git users and novices alike.
HDFView
A visual tool written in Java for browsing and editing HDF (HDF5 and HDF4) files.
ImageMagick
A free cross-platform software to create, edit, compose, or convert bitmap images.
OSGeo4W

A binary distribution of a broad set of open-source geospatial software (e.g., QGIS, GDAL/OGR, GRASS) for Windows environments.

(note: you can get Python3 w/ packages, netcdf, hdf5, and ffmpeg using OSGeo4W)

Pandoc
A universal document converter.
Panoply
A cross-platform application that plots geo-referenced and other arrays from netCDF, HDF, and other datasets.
QGIS 3
A free and open-source geographic information system.

Python Packages

exifread
Python package for reading exchangeable image file format (EXIF) metadata.1
geopy
A Python client for several popular geocoding web services.
gpsphoto
Return, modify, or remove GPS data from JPEG or TIFF Exif data.2 Requires ExifRead, piexif, and PIL.
h5py
Python package for reading and writing in the HDF5 binary data format.
pandas
A fast, powerful, flexible and easy to use open source data analysis and manipulation tool.
Pillow
A friendly PIL (Python Imaging Library) fork by Alex Clark and Contributors.
pyhdf
Python package for reading HDF4 (HDF-EOS) files.
scipy.io.netcdf
Provides modules, classes, and functions to read data from and write data to the classic NetCDF file format.
Unofficial Windows Binaries for Python Extension Packages
Provides 32- and 64-bit Windows (unofficial) binaries of many scientific open-source extension packages for the official CPython distribution of the Python programming language. A few binaries are available for the PyPy distribution.3

rmarkdown

Elegant, flexible, and fast dynamic report generation with R.
